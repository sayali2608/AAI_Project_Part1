This file lists the details about the folder structure.
1. dataset - sample dataset of 100 images.
2. saved_model - contains pre-saved model
3. test - contains image for application mode
4. test_samples - contains image for metrics
5. app.py - Application mode
6. Dataset_Provenance_Information - Information about datasets
7. Instructions on How to Run - How tu Run
8. Mask_Net.py - main file for training the model
9. metrics.py - file to generate test metrics
10. README - README file
11. References - File with Image reference information
12. Report.pdf - PDF Report
13. EOO.pdf - For Original work
